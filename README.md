# 🎨 だまし絵マスター (Trick Drawing Master)

「伝わりそうで伝わらない」絶妙なラインを狙う、オフライン環境で遊べるオンライン風お絵かき推理ゲームです。
1台のスマホやPCをプレイヤー同士で回しながら、ローカルでワイワイ遊ぶことができます。

## 🌟 ゲームのコンセプト

1人のプレイヤー（描き手）にだけ「お題」が表示されます。
描き手は、**「全員には伝わらないけれど、誰か1人・2人には伝わる」** ような、わざと分かりにくい絶妙な絵を描きます。
他のプレイヤー（回答者）はその絵を見て、お題が何かを予想して回答します。

描き手は他の人をうまくだませるほど有利になりますが、誰にも伝わらなさすぎるとペナルティになります！

## 🕹️ 特徴

* **1ファイル完結:** `index.html` のみで動作します。サーバー構築やデータベースは一切不要です。
* **レスポンシブ対応:** PCのブラウザはもちろん、スマートフォンのタッチ操作（スワイプでお絵かき）にも完全対応しています。
* **外部ライブラリ不使用:** Vanilla JS（ピュアなJavaScript）とHTML/CSSのみで構築されており、オフライン環境でも動作します。
* **自動採点システム:** ゲームのキモとなる「絶妙な絵を評価する」特殊なスコア計算を自動で行います。

## 📜 独自スコア・採点ルール

ゲームの面白さを引き出すため、以下の特殊な採点ルールが適用されます。

* **回答者:** 正解すると **+100点**
* **描き手:**
    * 正解者 1人 ➔ **+200点** （絶妙！最高得点）
    * 正解者 2人 ➔ **+150点** （なかなか絶妙）
    * 正解者 0人 ➔ **+80点** （誰にも伝わらなかった…）
    * 全員正解 ➔ **+30点** （簡単すぎた…）
    * ※その他の人数の場合は一律 +100点となります。

## 🚀 遊び方・セットアップ

1.  このリポジトリの `index.html` をダウンロードします。
2.  PCまたはスマートフォンのブラウザ（Chrome, Safari, Edgeなど）で `index.html` を開きます。
3.  タイトル画面でプレイヤー人数を設定し、名前を入力して「ゲームをはじめる」を押します。
4.  画面の指示に従って、現在の「描き手」に端末を渡し、お題を確認して絵を描きます。
5.  描き終わったら、他のプレイヤー（回答者）に順番に端末を渡し、他の人に見られないように予想を入力します。
6.  全員の予想が終わると正解発表と採点が行われます。
7.  全員が1回ずつ描き手を担当したら最終結果発表です！

## 🛠️ 技術スタック

* **HTML5:** Canvas APIを利用したお絵かき機能の実装
* **CSS3:** CSS変数（Custom Properties）を活用したポップなUIデザイン、Flexboxによるレスポンシブレイアウト
* **JavaScript (ES6+):** 状態管理、配列操作、Canvas描画ロジック、スコア計算アルゴリズム
